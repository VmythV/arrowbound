# arrowbound

arrowbound 是一款面向桌面浏览器的横版 2D 射箭增量 Roguelite 游戏。

玩家需要观察弓的摆动角度，选择时机射出具有真实重力轨迹的箭矢。命中靶子的不同环数会掉落不同价值的金币；金币可以用于永久商店升级、解锁机器人、培养金币宠物并推进关卡。

## 核心玩法

```text
选择关卡祝福
→ 把握弓的摆动时机射箭
→ 箭矢按抛物线飞行
→ 根据实际靶面交点判断环数
→ 拾取命中后掉落的金币
→ 购买永久升级
→ 花费目标金币完成关卡并解锁下一关
```

每关还可以主动开启 60 秒挑战。挑战成功后可以获得金币、免费商店升级或额外祝福候选等奖励。

## 首版 MVP

- 1280 × 720 横版 2D 游戏画面
- 玩家固定站位、弓自动摆动、点击射箭
- 真实重力抛物线和连续靶面碰撞检测
- 1～10 环判定、金币掉落和鼠标拾取
- 10 个完整配置关卡
- 永久商店升级和商品解锁树
- 每关首次进入时随机祝福 3 选 1
- 60 秒挑战和一次性挑战宝箱
- 机器人概率瞄准与真实物理射箭
- 金币宠物自动拾取
- 弹窗和浏览器隐藏时暂停
- localStorage 本地存档及异常恢复

## 技术栈

- Phaser 4.x
- TypeScript
- Vite
- localStorage

项目包含 Phaser 官方项目级技能，位于 `.agents/skills/`。开发 Phaser 功能时应加载对应技能，并优先采用 Phaser 4.x API。

技能来源：[phaserjs/phaser/skills](https://github.com/phaserjs/phaser/tree/master/skills)，当前导入版本为上游提交 `539e718ef0c19ddb0777645f2a470eccc8715691`。

首版只适配桌面网页。账号、云存档、排行榜、移动端和 3D 宝箱不属于当前 MVP。

## 关键设计原则

- 射箭手感优先于外围系统
- 射箭收益必须经过金币掉落和拾取流程后入账；宝箱奖励由奖励系统直接发放
- 机器人提供稳定收益，但不能取代玩家操作
- 手动命中 10 环的收益保持为机器人平均单箭收益的约 3～5 倍
- 关卡、商店、祝福、机器人和挑战全部配置化
- 奖励先写入待领取队列，再显示弹窗，避免刷新重复抽奖
- 所有视觉素材绘制为 SVG 文件并放入 `public/assets/svg/`
- 项目和游戏内容不使用 emoji，所有图标由 SVG 素材提供
- 动效必须兼顾即时反馈、信息清晰和重复游玩的节奏
- 开发前优先检查 TODOLIST，每完成一个任务立即更新其状态
