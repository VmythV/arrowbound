>
> 落地进度：§11 第 1（theme.ts + docs/07）、第 2（UI 配色/CTA/冷却条/**常驻目标进度条**/**三段胶囊 HUD**/弹窗全主题化）、第 3（角色/靶子萌化）、第 4（金币 5 档）、第 5（背景基线）已完成；爽感层已含十环全屏闪、达标脉冲、CTA 弹出、金币入账拖尾、**通关/挑战成功撒花**；展示字体已打包 **Baloo 2**（`public/assets/fonts/`，`main.ts` 启动前预加载，中文仍走 Noto Sans SC）。待续：宠物进化外观（Lv.10/20/30）、无限流其余主题背景换肤、更多关卡切换/宝箱开启动效打磨。

## 0. 一句话定位

把当前「黄昏博物馆」式的克制扁平画风，替换为**高饱和糖果色 + 粗描边胖萌造型 + 可爱吉祥物**的暖色卡通世界，并叠加一层**高对比爽感反馈**，让「命中十环 / 金币上涨 / 升级变强」的每个瞬间都值得期待。

## 1. 设计支柱

1. 命中即爽：十环、暴击、金币入账要有强色、强对比、强动势的正反馈。
2. 目标常在眼前：金币数旁常驻「距离通关目标」进度条，牵引「再射一箭」。
3. 世界温暖可爱：吉祥物角色 + 明亮世界，降低压力、鼓励长时间放置游玩。
4. 一眼可读：信息层级清晰，关键数值（金币、目标、冷却、倒计时）永远第一眼可见。

## 3. 字型系统

| 层级 | 字体 | 字重/字号 | 用途 |
|---|---|---|---|
| 展示体（数字/标题） | Baloo 2 或 Fredoka（圆润厚体） | 700，28–48px | 金币数、关卡名、弹窗标题、大数字弹出 |
| 中文正文 | Noto Sans SC | 500–700，16–22px | 说明、按钮、商品描述 |
| 数字强调 | 展示体 700 + 描边 | 40px+ | HUD 金币计数、命中「+N」弹出 |

替换现状：把标题的 `Georgia, serif` 换成圆体展示字；正文继续 Noto Sans SC。所有关键数字统一走「展示体 + 1px 深色描边 + 轻投影」，保证在明亮背景上也清晰。字体需本地打包，禁止依赖网络字体导致 FOUT。

## 4. 造型语言

- 描边：统一 5px（大物件 6px、小物件 4px），颜色用该物件暗部同色系而非纯黑。
- 圆角：一切转角圆润，UI 面板 20px、按钮 16px、卡片 18px。
- 体块：底部偏重的「胖萌」比例，重心低、圆润饱满。
- 明暗：两段平涂（亮面 + 暗面）+ 一道边缘高光（左上打光），不做复杂渐变。
- 高光：每个主体给一条白色 30% 透明的弧形高光，制造「果冻/陶瓷」质感。

## 5. 角色与吉祥物设计

统一走「大头、大眼、圆手、无复杂手指」的萌系比例。

- 玩家（小弓手）：圆脸大眼、鼓腮，头戴带小羽毛的软帽；拉弓时眼神专注、脸颊泛红点。三态：待机（帽羽轻摆）、拉弓（眯眼蓄势）、脱手（帽羽后甩 + 一道速度线）。
- 机器人（萌机械弓手）：圆角方体、单个发光大眼（呼吸闪烁）、小天线；命中时眼睛变成星形。发光眼颜色随机器人金币倍率提升可点亮更艳（成长可视化）。
- 金币宠物：圆滚滚的金币精灵，随等级进化——基础（小金币 + 圆眼）→ Lv.10 背小钱袋 → Lv.20 长小翅膀 → Lv.30 金色发光态。飞行拾取时留金色闪粉拖尾。
- 靶子：友好圆润的靶面 + 木质圆角靶架；命中十环时靶心「叮」地放光并轻微 Q 弹缩放，像被夸奖。

## 6. 世界与背景

### 6.1 分层视差

背景至少 4 层（远山雾 / 中景树丛 / 近景草坡 / 前景草叶），轻微视差随镜头微动，让静态场景有呼吸感。太阳/云做极缓慢漂移。

### 6.2 无限流换肤

配合已实现的无限流关卡（主题循环 + 周目）：每个主题一套「色相偏移 + 点缀物（树/石/晶体/雪松/熔岩）」，周目递增时叠加细节（如夜幕、萤火、星尘），让长期游玩视觉不重复。换肤只换背景与点缀，玩法几何不变。

## 7. 金币分层视觉

按价值分层的表现，对齐 `02_游戏玩法.md` §6.5：

| 价值 | 表现 | 视觉要点 |
|---|---|---|
| 1–3 | 小金币 | 单枚小币，微光 |
| 4–9 | 普通金币 | 标准币 + 星形高光 |
| 10–49 | 大金币 | 大币 + 轻描金边 |
| 50–199 | 金币袋 | 鼓胀钱袋 + 溢出币 |
| 200+ | 发光金币 | 强外发光 + 环绕闪粉 + 轻微上下浮动 |

价值越高，掉落时的弹跳与落地尘越夸张，拾取时飞向 HUD 的拖尾越亮。

## 8. UI 组件规范

坐标基于 1280×720 逻辑画布。图标均为 SVG，文中方括号为占位名。

### 8.1 顶部 HUD

分三组圆角胶囊，避免旧版「一整条深色板」：

```
[ (弓)第7关·暮色靶场 ]   [ (币)12,480  ▸ 目标 20,000 ]   [ (钟)挑战 0:48 ]  [ (花)金币祝福 ]
                          ▓▓▓▓▓▓▓▓▓░░░░░░  62%
```

- 左：关卡胶囊（关卡图标 + 第 N 关 + 关名）。
- 中（视觉主角）：大号金币计数 + 其下**目标进度条**（分段填充，达标时整条呼吸发光并出现「可通关」标）。这是留存的关键常驻牵引。
- 右：挑战倒计时（最后 10 秒变 `#FF6B5C` 并轻抖）+ 当前祝福胶囊（点击看效果）。

### 8.2 底部操作栏

厚实圆角按钮，图标 + 文字：

```
[ 商店 ]   [ 挑战 ]            (( 下一关 ▸ ))<脉冲发光>            [ 设置 ]
冷却 ▓▓▓▓▓▓░░░░  <达可射时整条微亮>
```

- 「下一关」为主 CTA：金币达标时用语义主色渐变 + 缓慢脉冲光召唤点击；不足时禁用灰并显示差额。
- 冷却条从旧的孤立轨道改为贴合底栏的扫描条；可射击瞬间整条闪一下作为「可以射了」的信号。

### 8.3 弹窗/模态

- 结构：柔和 scrim + 圆角卡片 + 顶部彩带标题条（标题居中 + 关闭按钮圆钮）。
- 出现：卡片从 92% 缩放 + 上浮淡入；scrim 同步淡入。
- 关闭：反向，快出（详细时序并入 07）。

### 8.4 商店卡片

每个商品一张卡：

```
┌───────────────────────────────┐
│ [道具大图标]  精准瞄准   Lv.7  │
│ 效果：十环基础半径 +7%         │
│ 等级 ●●●●●●●○○○                │
│           [ 价格 128 (币) ] [＋]│
└───────────────────────────────┘
```

- 够钱：价格胶囊变 `#4CD07D`、`＋` 按钮高亮可按。
- 不够：价格胶囊灰、显示差额。
- 已满级（受上限约束的道具）：显示「已满级」；无限流解封道具（贪婪金币/机械贪婪）永远显示下一级价格。
- 购买成功：卡片一圈金光 + 金粒飞溅 + 等级圆点点亮一颗。

### 8.5 祝福卡片（3 选 1 / 4 选 1）

- 魔法紫框圆角卡，图标 + 名称 + 效果一句话。
- 悬停轻浮起 + 描边发光；选定卡放大确认、其余卡飞出。

### 8.6 宝箱/奖励弹窗

- 卡通木箱，圆角胖箱盖；开启：箱盖弹跳翻开 + 强光柱 + 内容物（金币/升级/额外祝福）逐个跳出。
- 金币奖励用「发光金币」表现；升级奖励用对应道具图标；额外祝福用紫色祝福图标。

## 9. 关键爽感反馈层（Juice）

以下是**视觉与特效钩子**，具体触发时机、时长、缓动、可打断规则、加速行为等按 `07` 动效规范补全后再实现。所有高频反馈保持短促、不遮挡箭/靶/金币/按钮/倒计时。

| 时刻 | 反馈 |
|---|---|
| 命中普通环 | 命中点小火花 + 小「+N」上飘 + 靶面微弹 |
| 命中高环 | 金色火花放大 + 中号「+N」 |
| 命中十环（手动） | 全屏品红闪 + 白闪一帧 + 星爆 + 金币喷泉 + 大「+N!」弹跳 + 轻微镜头缩放（不震到眩晕） |
| 机器人命中 | 弱化版反馈，绝不盖过玩家手动十环（对齐现有不变量） |
| 脱靶 | 小尘土 + 箭插地轻颤，无惩罚感嘲讽 |
| 金币入账 | 金币拖尾飞入 HUD，金币计数弹跳一下 |
| 目标达标 | 进度条整条发光 + 「可通关」标出现 + 下一关按钮开始脉冲 |
| 关卡通关 | 彩带横幅 + 撒花 + 关名切换 |
| 挑战最后 10 秒 | 倒计时变红脉冲 + 边缘轻微暖光紧张感 |
| 挑战成功 | 定格光爆 + 宝箱登场 |

原则：玩家手动十环的反馈强度必须是全场峰值，自动系统（机器人/宠物）一律弱化，保证「亲手命中」的独占爽感。

## 10. SVG 资源清单（重画 / 新增）

对齐 `asset-manifest.ts` 分类。R=重画，N=新增。

- backgrounds：R `meadow-range`；N 其余 9 个主题背景 + 点缀物图层（无限流换肤）。
- characters：R `player-body`、`player-draw-arm`；N 待机/拉弓/脱手所需分件。
- bows / arrows：R `basic-bow`、`basic-bow-string`、`basic-arrow`（加粗描边、暖金箭羽）。
- targets：R `basic-target`（萌化 + 十环发光态）；N 靶架。
- coins：R 全部 5 档（small/basic/large/coin-bag/glowing），统一新风格与高光。
- robots：R `basic-robot`（独眼萌机器人 + 发光眼分件）。
- pets：R `basic-pet`；N 3 个进化态（钱袋/翅膀/发光）。
- chests：R `basic-chest`（胖箱 + 开启分件）。
- effects：R `hit-spark`、`miss-dust`；N `ten-ring-burst`（星爆）、`coin-trail`（闪粉拖尾）、`confetti`（撒花）、`glow-ring`（达标光环）。
- ui：R `hud-panel`、`modal-panel`、`blessing-card`、`cooldown-track/fill`、`loading-track/fill`、`modal-scrim`；N 胶囊底、进度条、商店卡底、图标集（商店/挑战/下一关/设置/关卡/时钟/祝福/金币）、CTA 发光态。
